# LIVE 소액 기술 검증 체크리스트

**목표**: 돈 벌기 ❌ / 시스템이 실전에서 안 터지는지 확인 ⭕

## 소액 LIVE 전 확인

- [ ] PAPER 모드에서 최소 1~2일 AUTO ON 방치 후, 주문/손절/이벤트가 정상 기록되는지 확인
- [ ] 대시보드에서 포지션/손익/이벤트/트레이드 목록이 실시간 반영되는지 확인
- [ ] 킬스위치(k 키 또는 UI KILL) 동작 확인

## LIVE 기술 검증 항목

| 항목 | 확인 |
|------|------|
| 주문이 실제로 들어간다 | |
| 체결/미체결 상태가 UI·DB에 정상 반영 | |
| 취소/재호가 정상 동작 | |
| 네트워크 끊었다가 재시작해도 포지션 복구 | |
| KILL 스위치 먹힘 (신규 주문 차단) | |

## 참고

- 이 단계에서 수익/손실 자체는 목표가 아님. “망가지는지”만 검증.
- 소액 한도는 `MAX_POSITION_KRW` 등 `.env`에서 설정.
